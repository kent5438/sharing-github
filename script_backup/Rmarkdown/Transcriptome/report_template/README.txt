Extension	Description
Trinity.fasta	(#COUNT1#) Raw assembled transcripts by Trinity.
Trinity.95.fasta	(#COUNT2#) Derived from raw assembled transcripts but clustered with 95% similarity.
Trinity.95.fasta.transdecoder.cds	(#COUNT3#) Nucleotide fasta of assembled transcripts.
Trinity.95.fasta.transdecoder.pep	Protein FASTA file of the translated CDS sequences.
Trinity.95.fasta.transdecoder.gff3	This is the master annotation in GFF3 format, containing both sequences and annotations. It can be viewed directly in Artemis or IGV.
Trinity.95.fasta.transdecoder.bed	BED (Browser Extensible Data) format provides a flexible way to define the data lines that are displayed in an annotation track.
