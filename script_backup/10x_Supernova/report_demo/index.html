FASTQs Demultiplexing Summary

The supernova mkfastq pipeline is a 10x-enhanced wrapper around Illumina bcl2fastq, which demultiplexes BCL files from a sequencer into FASTQs for analysis.

Assembly Summary

Supernova generates highly-contiguous, phased, whole-genome de novo assemblies from a Chromium-prepared library.

10x Supernova Assembly Process <— Tutorial

10x Supernova 4 Output of Assembly <— Tutorial

  1. raw: The raw style represents every edge in the assembly as a FASTA record. (seen as red segments in the cartoon). These include microbubble arms and gaps.

  1. megabubbles: In this style each megabubble arm corresponds to a FASTA record, as does each intervening sequence.

  1. pseudohap: The pseudohap style generates a single record per scaffold.

  1. pseudohap2.1 pseudohap2.2 : This style is like the pseudohap option, except that for each scaffold, two ‘parallel’ pseudohaplotypes are created and placed in separate FASTA files. Records in these files are parallel to each other.
